# 欢迎来到我的技术博客

这是我的第一篇博客文章。在这个博客中，我计划记录我的编程旅程，分享我作为软件工程师所学到的知识和见解。

## 为什么建立这个博客？

在信息爆炸的今天，碎片化的学习很容易让人迷失方向。通过撰写长文，我可以：
- **沉淀思考**：强迫自己系统性地梳理某个知识点。
- **知识共享**：帮助那些遇到同样问题的开发者。
- **职业档案**：展示我的技术栈和思维方式。

### 博客的技术架构

这个博客系统非常纯粹，它使用了：
1. **React 18**：最新的响应式 UI 框架。
2. **Tailwind CSS**：极致的样式开发体验。
3. **Markdown**：内容与表现分离。
4. **GitHub Actions**：自动化构建与部署。
